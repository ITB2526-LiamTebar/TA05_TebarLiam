Project 2 details: Data conversor.

Completion date: 17/11/2025.

General description:

This project consists in a code capable of extracting info from a Google form which then filters and displays. It was developed as a way to introduce myself into different type of documents such as CSV, XML, JSON and HTML.

Tech used for this project:

  • HTML5, CSS3 (Sass), JavaScript.
  • Python 3 (with libraries such as Pandas, JSON, XML, etree, ElementTree).
  • Google Forms & Google Sheets.
  • AIs, such as Google Gemini.

Completion and conclusions:

I really enjoyed making this project because I learned tons of new things, from filtering information to displaying, very basically, the results of the code. Making this wasn't easy at first asi I had no clue where to start but I caught up relatively fast thanks to aur friend Gemini over here...
